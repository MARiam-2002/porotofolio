# 🖼️ دليل إنشاء صورة معاينة الرابط (Open Graph Image)

## 📋 المتطلبات

### الأبعاد المطلوبة:
- **العرض:** 1200 بكسل
- **الارتفاع:** 630 بكسل
- **النسبة:** 1.91:1

### التنسيق:
- **الملف:** `og-image.jpg`
- **الحجم:** أقل من 1MB
- **الجودة:** عالية (80-90%)

## 🛠️ أدوات إنشاء الصورة

### 1. **Canva (مجاني)**
```
1. اذهب إلى canva.com
2. اختر "Custom Size"
3. أدخل: 1200 x 630
4. استخدم قوالب "Social Media"
5. صمم الصورة حسب المواصفات
```

### 2. **Figma (مجاني)**
```
1. اذهب إلى figma.com
2. أنشئ ملف جديد
3. اضبط الأبعاد: 1200 x 630
4. صمم الصورة
5. اصدّر كـ JPG
```

### 3. **Photoshop**
```
1. أنشئ ملف جديد
2. الأبعاد: 1200 x 630 pixels
3. صمم الصورة
4. احفظ كـ JPG
```

## 📤 رفع الصورة

### 1. **حفظ الملف**
```
اسم الملف: og-image.jpg
المسار: frontend/public/og-image.jpg
```

### 2. **اختبار الصورة**
```
1. اذهب إلى: https://developers.facebook.com/tools/debug/
2. أدخل رابط الموقع
3. تحقق من ظهور الصورة
4. اضغط "Scrape Again" إذا لزم الأمر
```

### 3. **اختبار WhatsApp**
```
1. شارك الرابط على WhatsApp
2. تحقق من ظهور الصورة
3. إذا لم تظهر، انتظر قليلاً أو أعد المحاولة
```

## 🔧 حل المشاكل

### إذا لم تظهر الصورة:

#### 1. **تحقق من الملف**
- تأكد من وجود الملف في المسار الصحيح
- تحقق من حجم الملف (أقل من 1MB)
- تأكد من صحة التنسيق (JPG)

#### 2. **تحقق من Meta Tags**
- تأكد من صحة الروابط في index.html
- تحقق من og:image URL
- تأكد من الأبعاد المذكورة

#### 3. **مسح الكاش**
- استخدم Facebook Debugger
- اضغط "Scrape Again"
- انتظر بضع دقائق

#### 4. **اختبار على منصات أخرى**
- Twitter Card Validator
- LinkedIn Post Inspector
- WhatsApp (شارك الرابط)
